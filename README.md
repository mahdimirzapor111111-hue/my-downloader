# 📥 دانلودر گیت‌هابی

با این پروژه می‌تونی هر فایلی رو با سرعت بالا دانلود کنی و توی مخزن گیت‌هاب ذخیره‌اش کنی.

## 📱 روش استفاده (کاملاً با گوشی)

1. برو به تب **Actions**
2. روی **دانلودر شخصی** کلیک کن
3. دکمه **Run workflow** رو بزن
4. لینک‌های دانلود رو وارد کن
5. منتظر بمون تا کار تموم بشه
6. فایل‌هات توی پوشه `downloads/` آماده‌اس

## ⚠️ نکات
- فایل‌های بزرگ‌تر از ۹۰MB خودکار تکه‌تکه میشن
- برای ترکیب فایل‌های تقسیم‌شده از `zip -F` استفاده کن
- فقط لینک مستقیم (Direct Link) قابل دانلوده
